# 🪂 ربات مانیتور ایردراپ

رباتی که کانال‌های تلگرام و اکانت‌های توییتر شما را رصد می‌کند و هر پست یا ایردراپ جدید را **با ترجمه فارسی** مستقیماً برایتان می‌فرستد.

> ✅ فقط با **Bot Token** کار می‌کند — نه my.telegram.org، نه شماره موبایل، نه هیچ دردسری!
> ✅ توییتر/X با GraphQL API رایگان (بدون کلید پولی) کار می‌کند!

---

## ✨ امکانات

| قابلیت | توضیح |
|--------|-------|
| 📢 رصد تلگرام | پیام جدید کانال‌ها را هر چند دقیقه چک می‌کند (اسکرپ مستقیم) |
| 🐦 رصد توییتر | پست‌های اکانت‌های X را هر چند دقیقه چک می‌کند (GraphQL API) |
| 🌍 ترجمه خودکار | هر پیام به فارسی ترجمه می‌شود |
| ➕ افزودن منبع | هر زمان منبع جدید اضافه کنید (`/add`) |
| ➖ حذف منبع | با یک کلیک منبع را حذف کنید |
| ⏸️ توقف موقت | هشدارها را موقتاً قطع/وصل کنید |
| 📊 وضعیت | آمار کامل ربات |

## 🚀 نصب و راه‌اندازی

### مرحله ۱ — نصب

```bash
cd airdrop_monitor
pip install -r requirements.txt
```

### مرحله ۲ — گرفتن Bot Token

1. در تلگرام **@BotFather** را باز کنید
2. `/newbot` بفرستید
3. یک اسم و یک یوزرنیم بدهید
4. **Bot Token** را کپی کنید (مثل `123456789:ABCdefGHIjkl...`)

### مرحله ۳ — تنظیمات

```bash
cp .env.example .env
```

فایل `.env` را باز کنید و فقط `BOT_TOKEN` را پر کنید:

```env
BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrSTUvwxYZ
```

> 🎉 همین! دیگر به هیچ چیز دیگری نیاز نیست.

### مرحله ۴ — اجرا

```bash
python bot.py
```

بعد از اجرا، در تلگرام ربات خود را باز کنید و `/start` بفرستید. 🎉

## 📋 دستورات

| دستور | کاربرد |
|-------|--------|
| `/start` | شروع ربات (اولین نفر = مالک) |
| `/menu` | نمایش منوی دکمه‌ای |
| `/add <لینک>` | افزودن منبع سریع |
| `/list` | لیست منابع |
| `/status` | وضعیت ربات |
| `/test` | ارسال پیام تست |
| `/help` | راهنما |

### افزودن منبع سریع
```
/add https://t.me/newchannel
/add https://x.com/username
```

یا از منوی ربات روی **➕ افزودن منبع** بزنید و لینک را بفرستید.

## ⚙️ تنظیمات پیشرفته (اختیاری)

فایل `.env`:

| متغیر | پیش‌فرض | توضیح |
|-------|---------|-------|
| `BOT_TOKEN` | — | توکن ربات (الزامی) |
| `RSSHUB_URL` | `https://rsshub.app` | آدرس RSSHub |
| `POLL_INTERVAL` | `300` | فاصله بررسی به ثانیه |
| `MAX_TRANSLATE_LENGTH` | `4500` | حداکثر طول متن ترجمه |

## 🔧 رصد چطور کار می‌کند؟

این ربات از دو روش مستقل برای رصد استفاده می‌کند:

### 📢 تلگرام
اسکرپ مستقیم از صفحه‌ی preview (`t.me/s/channelname`). بدون نیاز به API یا RSSHub.

### 🐦 توییتر / X
از GraphQL API داخلی توییتر با Guest Token استفاده می‌کند — **رایگان و بدون کلید پولی**.
این همان روشی است که کلاینت وب توییتر استفاده می‌کند.
- اگر rate-limit (429) شد، خودکار تلاش مجدد می‌کند
- نیازی به RSSHub یا Nitter نیست

---

## 🔒 امنیت

- فایل‌های `.env` و `data.db` حاوی اطلاعات حساس هستند — **هرگز به کسی ندهید**
- ربات فقط برای **یک نفر** (اولین کسی که `/start` می‌زند) کار می‌کند
- این ربات با اکانت تلگرام شما هیچ ارتباطی ندارد — کاملاً مستقل است

---

## ❓ عیب‌یابی

| مشکل | راه‌حل |
|------|--------|
| `BOT_TOKEN خالی است` | فایل `.env` را پر کنید |
| ربات جواب نمی‌دهد | مطمئن شوید `/start` زده‌اید |
| ترجمه کار نمی‌کند | اتصال اینترنت را چک کنید |
| توییتر ❌ می‌شود | rate-limit موقتی است؛ ربات خودکار retry می‌کند |
| تلگرام ❌ می‌شود | کانال ممکن است خصوصی باشد یا نام تغییر کرده باشد |
| `Conflict: terminated by other getUpdates` | ربات قبلاً در حال اجراست؛ قبلی را ببندید |
